JS+CSS实现鼠标滚动透明和移动

代码是在HTML文档的`DOMContentLoaded`事件触发时执行的。当文档内容加载完成时，它会获取一个ID为`bttm`的元素，并为其添加一个滚动事件监听器。 实现原理是，当用户在滚动时，会检查`bttm`元素是否在视口内。如果在视口内，则使其可见，否则使其不可见。这样可以实现一个简单的滚动动画效果。
